Добро пожаловать на сайт электронной редакции издательства «Медиа Сфера»!
Чтобы отправить статью в один из журналов издательства, необходимо зарегистрироваться. Для этого нажмите кнопку РЕГИСТРАЦИЯ в верхней навигационной панели и заполните форму. Если вы уже зарегистрировались, введите имя пользователя и пароль для входа в систему. В разделе «Материалы для авторов» вы найдете подробную инструкцию по пользованию электронной редакцией.
